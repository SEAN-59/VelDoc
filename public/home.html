도우미

폴더 선택 기준

열기에서는 문서를 저장할 프로젝트 폴더를 선택합니다
선택한 폴더 안에 VelDoc이 사용할 veldoc 폴더가 생성됩니다

선택하는 위치
문서를 모아둘 최상위 작업 폴더입니다
예를 들어 ProjectA를 선택하면 ProjectA/veldoc 아래에 문서 폴더가 생깁니다

버튼 동작

각 버튼은 필요한 문서 폴더를 먼저 확인합니다
폴더가 없으면 새로 만들고 이미 있으면 그대로 연결합니다

API 명세서
veldoc/api를 준비한 뒤 API 명세서 에디터로 이동합니다
DB 테이블 정의서
veldoc/table을 준비한 뒤 DB 테이블 정의서 에디터로 이동합니다
나머지 문서 버튼
준비 중인 문서 작성기입니다

생성 구조

veldoc/wbs
WBS 문서 폴더입니다
veldoc/srs
요구사항 정의서 문서 폴더입니다
veldoc/fsd
기능 명세서 문서 폴더입니다
veldoc/api
API 명세서 문서 폴더입니다
veldoc/table
DB 테이블 정의서 문서 폴더입니다

세션 유지

브라우저 권한이 유지되는 동안 선택한 폴더를 다시 사용합니다
권한이 만료되거나 다른 브라우저를 쓰면 열기에서 다시 폴더를 허용해야 합니다

개발 문서를 정리하는 가장 귀여운 방법

※ VelDoc 폴더를 만들 상위 폴더를 선택하세요.
