저장 / 파일 위치

홈에서 먼저 작업 폴더를 선택해야 WBS 문서를 작성하고 저장할 수 있습니다.
WBS 문서는 선택한 폴더의 veldoc/wbs 안에 Markdown으로 저장됩니다.

파일 목록
왼쪽 파일 목록에서 저장된 WBS 문서를 선택해 다시 열 수 있습니다.
파일명
새 파일은 프로젝트명-내부코드-wbs.md 형식으로 저장됩니다.
중복
같은 이름의 파일이 이미 있으면 저장하지 않고 토스트로 알려줍니다.

작성 가이드

WBS는 프로젝트 목표를 실제 수행 가능한 작업 단위로 나누는 문서입니다.
프로젝트명, 내부 코드, 버전, 담당자는 필수로 입력합니다.
작업 목록은 1개 이상 필요합니다.

WBS 번호
1, 1.1, 1.1.1처럼 숫자와 점으로만 작성합니다.
작업
실제로 수행할 일을 명확한 동사나 산출물 중심으로 적습니다.
들여쓰기
WBS 번호의 깊이에 맞춰 카드 위치가 자동으로 정리됩니다.

날짜 / 진행률

작성일
새 문서를 만들 때 자동으로 들어갑니다.
수정일
저장할 때마다 저장 날짜로 자동 변경됩니다.
기간
시작일과 종료일을 포함한 전체 일수입니다.
영업 기간
주말과 한국 주요 공휴일을 제외한 기준 일수입니다.
진행률
영업 기간 기준으로 자동 계산하고 100%를 넘기지 않습니다.
상위 작업
하위 작업이 있으면 시작일과 종료일을 하위 일정 기준으로 자동 계산합니다.
